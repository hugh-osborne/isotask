Data Collected by Piyanee Shriya, Sarah Askill, and Samit Chakrabarty at University of Leeds, UK.

Please see [1](https://doi.org/10.1152/jn.00208.2021) for a full description of the protocol.

Each data file is in CSV format with the following columns quoted on the first row:

Times
RF (Rectus Femoris)
VL (Vastus Lateralis)
VM (Vastus Medialis)
ST (Semitendinosus)
BF (Biceps Femoris)
MG (Medial Gastrocnemius)
TA (Tibialis Anterior)

The data files are duplicated across two different directory structures for convenience:

<Position> / <Internal Knee Angle> / <Participant>
<Position> / <Participant> / <Internal Knee Angle>

emg.py can be run to analyse the data and produce the figures used in the paper. 

Contact Samit Chakrabarty (s.chakrabarty@leeds.ac.uk) for permissions and further information.

[1] York, G., Osborne, H., Sriya, P., Astill, S., de Kamps, M. and Chakrabarty, S., 2022. The effect of limb position on a static knee extension task can be explained with a simple spinal cord circuit model. Journal of Neurophysiology, 127(1), pp.173-187.
